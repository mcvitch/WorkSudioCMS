 - Criar um repositório: git init
 - Adicionar alterações no git (STAGE): git add <<nome do arquivo>> ou git add. pra adicionar todos
 - Verifiar se tem algum arquivo que não foi enviada (STAGE): git status
 - criar uma nova versão - enviar o arquivo ao git (COMMIT - aplicar uma alteração ao sistema): git commit -m
